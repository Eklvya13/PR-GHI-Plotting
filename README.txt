How to Run

Setup >>
Place the extracted dataset folder "Data" in the same directory as main.py.

Running the Script >>
The script will preprocess the data and generate a combined.csv file. This file contains the cleaned data and is used to create the plot.

Output >>
combined.csv: Preprocessed data.
plot_daterange.jpeg: Generated plot image.


Additional Files >>
PreprocessedData.csv: Preprocessed data sample.
SampleDatasetPlot.jpeg: Example plot image.
